Image Alignment

The CSS variables for this feature are deprecated. Use the image-align (horiztonal alignment) and the image-valign (vertical alignment) attributes instead.

More CSS Variables

CSS Variables
Variable Name Description
--accent-card-color Sets the accent card text color, overriding the colors set by accent-color and dark attributes.
--accent-card-background-color Sets the accent card background color, overriding the colors set by accent-color and dark attributes.
--accent-card-border-color Sets the accent card border color, overriding the colors set by accent-color and dark attributes.
--accent-card-box-shadow Sets the box shadow of the accent card, unless the card has the flat attribute.
--accent-card-heading-color Sets the heading text color, overriding the colors set by accent-color and dark attributes.
--accent-card-footer-border-color Sets the color of the border above the footer, overriding the colors set by accent-color and dark attributes.
--accent-card-padding Sets padding inside the accent card.

CSS Mixins

CSS Mixins
Mixin Name Scope
--accent-card Applies to the entire accent card.
--accent-card-body Applies to the body side of the card.
--accent-card-content Applies to the content slot of the card body.
--accent-card-footer Applies to the footer slot of the card body.
--accent-card-heading Applies to the heading slot of the card body.
--accent-card-subheading Applies to the subheading slot of the card body.
--accent-card-image Applies to the accent card, regardless of orientation.
--accent-card-image-horizontal Applies to the accent cards with the horizontal attribute.
--accent-card-image-vertical Applies to the accent cards with the default vertical orientation.
